# Worked Examples

[**← repository root**](../README.md) &nbsp;·&nbsp; `examples/` — the generic templates made concrete

---

Each directory here is a **complete concrete adaptation** of the generic templates for one
stack — zero placeholders, real commands, realistic project conventions. Currently:

- [`laravel-react/`](laravel-react/) — Laravel 12 API + React SPA monorepo (the stack closest
  to the methodology's production origin).

> [!TIP]
> **PRs adding `examples/<your-stack>/` are the most valuable contribution this repository
> accepts.** Follow the structure of `laravel-react/` (README with a placeholder→value mapping
> table and an honesty note, the nine concrete agents, a condensed CLAUDE.md) and the
> sanitization rules in [CONTRIBUTING.md](../CONTRIBUTING.md) — nothing from your private
> systems may leak into your example.
